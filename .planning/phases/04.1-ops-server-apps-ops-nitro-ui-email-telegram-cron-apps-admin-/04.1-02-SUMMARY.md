## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Missing `relaySecret` in ops runtimeConfig**
- **Found during:** Task 4 (typecheck after copying auth.ts)
- **Issue:** `requireRelaySecret()` в `auth.ts` вызывает `useRuntimeConfig().relaySecret?.trim()`, но `relaySecret` не был объявлен в `runtimeConfig` в `nuxt.config.ts`. TypeScript strict mode выдавал `error TS2339: Property 'trim' does not exist on type '{}'` (вывод `relaySecret` как тип `{}`).
- **Fix:** Добавлен `relaySecret: ''` в runtimeConfig ops/nuxt.config.ts с комментарием `// NUXT_RELAY_SECRET — для requireRelaySecret() (PREPROD-212 defense-in-depth)`
- **Files modified:** `apps/ops/nuxt.config.ts`
- **Commit:** `6e41c233` (включён в Task 4 commit)
