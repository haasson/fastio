# Wishlist — идеи и хотелки

## Редактор зон доставки в adminке

**Суть:** Владелец заведения сам рисует зоны доставки на карте прямо в настройках. Без внешних инструментов, без техподдержки.

**Как выглядит:**
- Страница настроек доставки содержит интерактивную карту
- Можно создать несколько зон (полигонов), у каждой свои условия
- Для каждой зоны задаётся: название, цвет, цена доставки, минимальная сумма заказа, время доставки
- Зоны можно редактировать и удалять
- Карта центрируется по адресу заведения

**Технически:**
- БД: таблица `delivery_zones` — id, branch_id, name, color, polygon (GeoJSON или массив координат), delivery_price, min_order, delivery_time_min
- Supabase поддерживает PostGIS — можно `ST_Contains(zone, point)` прямо в SQL для проверки адреса
- Карта: Yandex Maps API (для РФ) или Leaflet + OpenStreetMap (бесплатно)
- Редактор полигонов: встроенный инструмент рисования (Yandex Maps умеет, Leaflet.draw плагин)
- Геокодинг на чекауте: адрес → координаты → проверка по зонам
- Публичная страница заведения: показывать зоны на карте (как у angry-pizza.ru/delivery)

**Что нужно сделать:**
1. Миграция БД: таблица `delivery_zones`
2. API: CRUD для зон
3. Adminка: страница/секция редактора зон с картой
4. Чекаут: валидация адреса по зонам, подтягивание условий
5. Публичная часть: карта с зонами на странице доставки заведения

## Кроппер изображений при загрузке

**Суть:** При загрузке картинки показывать кроппер, где юзер сам выбирает какая часть изображения попадёт в финальную версию. Загружается уже обрезанная картинка нужного размера/ratio.

**Как выглядит:**
- Юзер выбирает файл → открывается модалка с кроппером
- Фиксированная рамка нужного aspect ratio (разная для товаров, категорий, баннеров и т.д.)
- Юзер двигает и масштабирует картинку под рамку
- Жмёт "Применить" → в Supabase Storage улетает уже обрезанный blob

**Технически:**
- Библиотека: `vue-advanced-cropper` (Vue 3 native, touch-friendly) или `cropperjs`
- `canvas.toBlob()` для получения обрезанного файла перед upload
- Нужно определить нужные ratio/размеры для каждого типа изображений (товары, категории, баннеры)
- Хранить можно только обрезанное, или оригинал + обрезанное (для будущего перекадрирования)
