## Меню
- **URL:** /menu
- **Пермишен:** menu.view (просмотр), menu.edit (редактирование), menu.delete (удаление)
- **Описание:** Управление каталогом — категории, блюда/товары, цены, фото

### Блюда
- **URL:** /menu/dishes
- **Действия:**
  - Кнопка «Добавить» → открывается форма «Новое блюдо» (название, цена, описание, фото, категория, вес/объём)
  - Клик по блюду → форма «Редактировать блюдо»
  - Удалить блюдо (кнопка «Удалить» в форме)
  - Поставить в стоп-лист — тогл «Активен» в карточке блюда
  - Перетаскивание для сортировки
  - Поиск: поле «Поиск блюда…»
- **Пустое состояние:** отображается при отсутствии блюд в категории

### Категории
- **URL:** /menu/categories
- **Действия:**
  - Кнопка «Добавить категорию» → форма «Новая категория»
  - Клик → форма «Редактировать категорию» (название, описание)
  - Кнопка «Удалить» — только если нет блюд внутри. Если есть: «В этой категории есть блюда. Сначала переместите их в другую категорию.»
  - Перетаскивание для сортировки
  - Привязка к филиалам
- **Пустое состояние:** «Категорий пока нет. Создайте первую — она появится в меню.»

### Модификаторы
- **URL:** /menu/modifiers
- **Модуль:** modifiers
- **Описание:** Группы опций для блюд (размер, прожарка, тип теста и т.д.)
- **Действия:**
  - Кнопка «Добавить группу» → форма «Новый модификатор» (название группы, опции, min/max выбор)
  - Кнопка «Добавить опцию» внутри группы
  - Привязать к блюдам
  - Удалить — подтверждение: «Группа «{name}» будет удалена. Привязки к блюдам тоже исчезнут.»
- **Пустое состояние:** «Модификаторов пока нет. Создайте первую группу — например «Размер» или «Бортик».»

### Добавки
- **URL:** /menu/addons
- **Модуль:** addons
- **Описание:** Дополнительные товары к заказу (соусы, напитки)
- **Вкладки:** «Добавки» и «Пресеты»
- **Действия (вкладка «Добавки»):**
  - Кнопка «Добавить» → форма «Новая добавка» (название, цена)
  - Поиск: «Поиск по названию…»
  - Удалить — подтверждение: «{name}» будет удалена. Привязки к блюдам тоже исчезнут.»
- **Действия (вкладка «Пресеты»):**
  - Кнопка «Добавить» → форма «Новый пресет»
  - Пресеты позволяют быстро назначать наборы добавок на блюда
  - Удалить — подтверждение: «Пресет «{name}» будет удалён.»
- **Настройка:** тогл «Ограничить количество добавок на блюдо»
- **Пустое состояние:** «Добавок пока нет. Создайте первую — например «Халапеньо» или «Соус барбекю».»

### Теги
- **URL:** /menu/tags
- **Описание:** Метки для блюд и виртуальные категории на основе тегов
- **Действия:**
  - Кнопка «Добавить тег» → форма «Новый тег»
  - Клик → форма «Редактировать тег»
  - Перетаскивание для сортировки
  - Виртуальная категория автоматически показывает все блюда с выбранным тегом из любых категорий
- **Пустое состояние:** «Тегов пока нет. Создайте первый — он поможет фильтровать позиции.»
