## Контент
- **URL:** /content
- **Пермишен:** content.view (просмотр), content.edit (редактирование)
- **Описание:** Управление контентом публичного сайта

### Баннеры
- **URL:** /content/banners
- **Действия:**
  - Кнопка «Добавить баннер» → форма (изображение, действие при клике: акция / промокод / страница / ссылка / ничего не происходит)
  - Перетаскивание для сортировки
  - Включить/выключить баннер
  - Удалить баннер
  - Предупреждения о битых ссылках (если привязанная акция/промокод были удалены)
- **Пустое состояние:** «Баннеров пока нет. Добавьте первый — он появится в карусели на главной.»

### Галереи
- **URL:** /content/gallery
- **Действия:**
  - Кнопка «Добавить галерею» → форма (название)
  - Кнопка «Фото» — управление фотографиями в галерее
  - Добавить/удалить фото, перетаскивание для сортировки
  - Тег «Не используется» — если галерея не привязана к секции/странице
  - Счётчик «{N} фото»
  - Удалить галерею
- **Пустое состояние:** «Галерей пока нет. Создайте первую — потом выберете её в настройках разделов.»
