## Акции
- **URL:** /promotions
- **Модуль:** promotions
- **Пермишен:** promos.view (просмотр), promos.manage (управление)
- **Описание:** Скидки, промокоды и спецпредложения

### Акции
- **URL:** /promotions/list
- **Действия:**
  - Кнопка «Добавить» → форма (название, тип акции, тип скидки, размер скидки, мин. сумма заказа, период действия)
  - Поиск: «Поиск по названию...»
  - Тогл «Активна» — включить/выключить акцию
  - Редактировать, удалить
- **Пустое состояние:** «Акций пока нет. Создайте первую — например скидку в happy hour.»

### Промокоды
- **URL:** /promotions/promo-codes
- **Действия:**
  - Кнопка «Добавить» → форма (код, тип скидки, размер, мин. сумма, лимит использований, период действия)
  - Поиск: «Поиск по коду...»
  - Тогл «Активен» — включить/выключить
  - Редактировать, удалить
- **Пустое состояние:** «Промокодов пока нет. Создайте первый — например WELCOME10 на скидку 10%.»
