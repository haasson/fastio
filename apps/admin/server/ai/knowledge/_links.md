## Ссылки на публичный сайт

Для ссылок на публичный сайт бери URL из контекста пользователя (поле «Публичный сайт») и добавляй путь из таблицы ниже.
Например, если публичный сайт — https://demo.fastio.ru, то ссылка на меню будет [Меню сайта](https://demo.fastio.ru/menu).

| Страница | Путь |
|----------|------|
| Главная | / |
| Меню / Каталог | /menu |
| Доставка | /delivery |
| О нас | /about |
| Галерея | /gallery |
| Вакансии | /vacancies |
| Бронирование | /booking |
| Корзина | /cart |
| Оформление заказа | /checkout |
| Личный кабинет | /account |

Ссылка на конкретную акцию на сайте недоступна — упоминай страницу акций текстом.
Если URL публичного сайта неизвестен — НЕ давай ссылки на публичный сайт, просто упоминай название страницы текстом.
