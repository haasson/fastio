## Столы
- **URL:** /tables
- **Модуль:** dineIn
- **Пермишен:** tables.view (просмотр), tables.manage (управление)
- **Описание:** Управление столами для заказа в зале

### Список столов
- **URL:** /tables/list
- **Секции:** «Открытые» и «Закрытые»
- **Действия:**
  - Кнопка «Новый стол» → форма (название, кол-во мест, доп. информация)
  - Карточки столов: номер, сумма сессии, количество позиций
  - Кнопка «Открыть стол» — начать сессию
  - Кнопка «+ Блюдо» — добавить блюдо к столу
  - Кнопка «Расчёт» → модал «Расчёт: {название стола}» (скидка %, ₽, кнопка «Закрыть стол»)
  - Индикатор «блюдо готово / блюд готово к подаче» — кнопка «Забрал»
  - Кнопка QR-код → показать/скачать QR для заказа со стола
  - Кнопка «Редактировать» / «Готово» — режим редактирования
  - Кнопка «Удалить» в режиме редактирования
- **Пустое состояние:** «Столов пока нет»

### Схема зала
- **URL:** /tables/layout
- **Действия:**
  - Интерактивная карта с расположением столов
  - Перетаскивание столов на карте
  - Добавление блюд, расчёт, бронирование прямо с карты

### Вызовы
- **URL:** /tables/calls
- **Действия:**
  - Просмотр активных вызовов от гостей
  - Поле «Название типа вызова» — добавить новый тип
  - Отметка вызова как обработанного
- **Пустое состояние:** «Нет активных вызовов»
