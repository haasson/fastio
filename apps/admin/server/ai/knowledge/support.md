## Поддержка
- **URL:** /help/support
- **Описание:** Система обращений в поддержку

### Создание обращения
- Кнопка «Создать обращение» → форма: тема, описание проблемы, до 3 изображений
- После создания — обращение появляется в списке слева

### Переписка
- Выбрать обращение из списка → справа открывается чат
- Сообщения от поддержки (слева) и от вас (справа) в виде чата
- Отправка текста и изображений
- Индикатор непрочитанных сообщений (счётчик)

### Статусы
- **Открыт** (жёлтый) — новое обращение или ваш ответ
- **Ожидает ответа** (синий) — поддержка ответила, ждём вашу реакцию
- **Закрыт** (серый) — решён. Автозакрытие через 5 дней без ответа. Можно переоткрыть, просто написав новое сообщение.

### Где следить
- Список обращений: [Поддержка](/help/support)
- Счётчик непрочитанных в боковом меню

## Виртуальные туры
- **URL:** /help/tours
- **Описание:** Интерактивные пошаговые обучающие туры по разделам админки

### Доступные туры

**Начало работы:**
- «Обзор интерфейса» — быстрая экскурсия по всем разделам админ-панели

**Меню:**
- «Как создать категорию» — пошаговый тур по созданию категории меню
- «Как создать блюдо» — добавление блюда в меню за несколько шагов
- «Как работают модификаторы» — создание групп вариантов (размер, прожарка и т.д.)
- «Как работают добавки» — соусы, топпинги, пресеты для быстрого назначения

**Заказы:**
- «Как работать с заказами» — статусы, карточки заказов, быстрая смена статуса и создание заказа вручную

### Как запустить тур
- Перейти в [Виртуальные туры](/help/tours)
- Выбрать нужный тур из списка
- Нажать кнопку «Запустить»
- Тур проведёт по шагам прямо в интерфейсе с подсказками
