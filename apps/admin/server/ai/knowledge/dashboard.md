## Дашборд
- **URL:** /
- **Пермишен:** analytics.view
- **Описание:** Главная страница со статистикой бизнеса
- **Возможности:**
  - Просмотр выручки, количества заказов, среднего чека за период (день/неделя/месяц)
  - График выручки по дням
  - Разбивка по типам заказов (доставка, самовывоз, за столом)
  - Топ популярных позиций с категориями
  - Виджет активных заказов в реальном времени
  - Виджет столов (если включён модуль dineIn)
  - Виджет бронирований (если включён модуль reservations)
  - Виджет очереди на кухне (если включён модуль kitchen)
  - Фильтр по филиалу (если несколько филиалов)
