## Кухня
- **URL:** /kitchen
- **Модуль:** kitchen
- **Пермишен:** kitchen.view, kitchen.overview
- **Описание:** Экран для повара — заказы в реальном времени

### Очередь
- **URL:** /kitchen/queue
- **Описание:** Список блюд для приготовления
- **Действия:**
  - Фильтр «Мои категории (все)» — показать только нужные категории
  - Секция «Очередь» — блюда, ожидающие приготовления
  - Кнопка «Взять» — забрать блюдо себе в работу
  - Секция «Мои блюда» — то, что повар взял в работу
  - Кнопка «Готово» — блюдо готово
  - Кнопка «Вернуть» — вернуть блюдо в очередь
  - Кнопка «Убрать» — убрать из своего списка
  - Карточки подсвечиваются жёлтым на 66% и красным на 100% порога срочности
- **Пустое состояние:** «Нет блюд в этой категории»
- **Предупреждения:**
  - «Не настроен статус для отправки заказов на кухню» → кнопка «Настроить» ведёт в настройки
  - «Не настроены статусы завершения для: …»

### Сборка
- **URL:** /kitchen/assembly
- **Описание:** Сборка заказов для доставки/самовывоза
- **Действия:**
  - Чекбоксы у каждой позиции
  - Кнопка «Собрано» — отметить заказ собранным

### Обзор
- **URL:** /kitchen/overview
- **Описание:** Обзор всех заказов на кухне

### Настройки кухни
- **URL:** /kitchen/settings
- **Действия:**
  - «Статус для отправки на кухню» — выбрать, при каком статусе заказа блюда попадают в очередь
  - «Статус при начале готовки или сборки» — автосмена при взятии блюда
  - «Собрано: доставка / Собрано: самовывоз» — статус при завершении сборки
  - «Порог срочности (минуты)» — время до подсветки карточек
  - Кнопка «Сохранить»
