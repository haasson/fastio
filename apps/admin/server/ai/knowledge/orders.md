## Заказы
- **URL:** /orders
- **Пермишен:** orders.view (просмотр), orders.edit (редактирование), orders.status (смена статуса), orders.cancel (отмена)
- **Описание:** Приём и управление заказами
- **Требует:** хотя бы один модуль заказов (delivery, pickup, или dineIn)

### Список заказов
- **URL:** /orders
- **Действия:**
  - Кнопка «Новый заказ» → форма создания (имя, телефон, филиал, адрес, дата/время, комментарий)
  - Фильтрация по статусу (вкладки сверху с подсчётом)
  - Поиск: «Имя или телефон…»
  - Массовые действия: выбрать заказы → «Сменить статус…» → «Применить»
  - Кнопка «Экспорт CSV» для выгрузки
  - Клик по заказу → детали
- **Пустое состояние:** «Заказов пока нет»

### Детали заказа
- **URL:** /orders/[id]
- **Действия:**
  - Просмотр полной информации
  - Редактирование заказа (кнопка «Сохранить»)
  - Смена статуса
  - Поле «Написать заметку…» для внутренних комментариев

### Статусы заказов
- **URL:** /orders/statuses
- **Действия:**
  - Кнопка «Добавить статус» → форма (название, быстрые действия)
  - Редактирование, удаление статусов
  - Каждый статус может иметь до 2 быстрых действий
- **Пустое состояние:** «Нет статусов»

### Настройки заказов
- **URL:** /orders/settings
- **Действия:**
  - Включить/выключить предзаказы
  - Настроить количество доступных дней (1-30)
  - Шаг слотов (15, 30, 60 мин)
  - Буфер закрытия (за сколько до закрытия прекращать приём)
  - Время подготовки доставки/самовывоза

### Нумерация заказов
- **URL:** /orders/order-number
- **Действия:**
  - Формат номера: счётчик, префикс+счётчик, дата+счётчик, всё вместе
  - Область нумерации: глобальная или по филиалам
  - Настройка префикса, формата даты, сброса, начального номера
  - Предпросмотр формата в реальном времени

### Настройки доставки
- **URL:** /orders/delivery
- **Модуль:** delivery
- **Действия:**
  - Режим: фиксированная стоимость или зоны доставки
  - Зоны доставки на карте: создание зон, привязка к филиалам, стоимость/минимальный заказ/бесплатная доставка по зонам
