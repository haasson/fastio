# Дашборд

Главная страница панели управления — ваш командный центр. Здесь собрана вся ключевая информация о работе заведения в одном месте.

## Статистика и период

В верхней части страницы выберите период: **Сегодня**, **Неделя** или **Месяц**. Все показатели пересчитываются автоматически.

Если у вас несколько филиалов, рядом появится переключатель филиала — владелец может смотреть сводку по всем или только по конкретному.

**Карточки показателей:**
- **Выручка** — сумма завершённых заказов за период
- **Заказов** — количество оформленных заказов
- **Средний чек** — выручка ÷ количество заказов

## Графики

**График выручки** показывает динамику по дням внутри выбранного периода — удобно замечать провалы и пики.

**Распределение по типам** — доля доставки, самовывоза и зала. Помогает понять, какой канал работает лучше.

**Топ-10 позиций** — самые продаваемые блюда за период. Можно фильтровать по категории.

## Виджеты реального времени

Внизу страницы — живые счётчики, которые обновляются без перезагрузки:

- **Активные заказы** — заказы в работе прямо сейчас (не завершённые и не отменённые)
- **Открытые столы** — столы с активными сессиями (если включён модуль «Столы»)
- **Ближайшие бронирования** — брони на сегодня (если включён модуль «Бронирования»)
- **Кухня** — позиции в очереди на приготовление (если включён модуль «Кухня»)

## Онбординг

Если вы только что создали заведение и ещё не включили ни один модуль заказов — вместо статистики появится мастер первоначальной настройки. Он проведёт вас по ключевым шагам: настройка меню, подключение модулей, оформление витрины.

## Связи с другими разделами

- Клик по счётчику активных заказов → переход в **Заказы**
- Открытые столы → переход в **Столы**
- Ближайшие бронирования → переход в **Бронирования**
- Кухня → переход в **Кухня → Очередь**
