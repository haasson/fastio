# Работа с заказами

Здесь поступают все входящие заказы — доставка, самовывоз и столы. Страница обновляется в реальном времени: новые заказы появляются без перезагрузки.

## Интерфейс списка

Вкладки вверху — это статусы заказов. На каждой вкладке показано количество заказов в этом статусе. Кликните на вкладку — увидите заказы в этом статусе.

**Переключение вида:** кнопки «Карточки» / «Таблица» в правом верхнем углу.
- **Карточки** — удобно для работы на кассе, все данные сразу видны
- **Таблица** — удобно для аналитики, можно настроить колонки

Ширина карточек настраивается в **Заказы → Настройки → Отображение заказов** (компактные / средние / крупные); карточки растягиваются на всю ширину ряда.

**Поиск:** поле «Имя или телефон…» — ищет по имени клиента и номеру телефона в рамках выбранного статуса.

**Фильтры в таблице:** по типу доставки (доставка / самовывоз), по способу оплаты (наличные / карта), по филиалу.

## Карточка заказа

На каждой карточке:
- Номер заказа и текущий статус (цвет совпадает с группой статуса)
- Тип: «Доставка» / «Самовывоз» / название стола
- Время создания (относительное — «5 мин назад», при наведении — точное)
- Имя и телефон клиента (по телефону можно позвонить кликом)
- Адрес (для доставки)
- Состав: до 3 позиций, остальные скрыты под «+X ещё»
- Промокод или акция (если применялись)
- Итоговая сумма и способ оплаты
- **Тег запланированного времени** (для предзаказов) — «Сегодня 14:00», «Завтра 10:30» или «25.04 14:00» с иконкой часов
- **Кнопки быстрого перехода** — до 2 кнопок для смены статуса (настраиваются в разделе Статусы). Для запланированных заказов скрываются, пока заказ не переведён в статус «Запланировано» — это защита от случайной отправки на кухню раньше времени.

## Открытие заказа

Кликните по карточке — откроется детальная страница заказа со всей информацией и возможностью редактирования.

## Создание заказа вручную

Кнопка «+ Новый заказ» в правом верхнем углу.

### Поля формы

**Клиент:**
- Имя (необязательно)
- Телефон (обязательно)

**Тип заказа:** Доставка / Самовывоз / Со стола

**Для доставки — адрес:**
- Основной адрес с автоподсказками (DaData)
- Подъезд, Этаж, Квартира, Домофон
- Система автоматически определяет зону доставки и показывает стоимость

**Время:**
- «Как можно скорее» — стандартный заказ
- «К определённому времени» — предзаказ. Выберите дату и время из доступных слотов (слоты фильтруются по рабочим часам, буферу закрытия и минимальному времени подготовки)
- **За сколько готовить (слайдер)** — доступен только для предзаказов. Сколько минут до назначенного времени заказ должен попасть на кухню. От 15 минут до 3 часов, по умолчанию 60 минут. Например: заказ к 14:00, слайдер 60 минут → кухня получит заказ в 13:00.

**Состав заказа:**
- Добавляйте блюда кнопкой «+ Добавить блюдо»
- В пикере можно выбрать модификаторы, убрать ингредиенты, выбрать добавки

**Скидка/промокод:** выберите из списка доступных акций или введите промокод

**Оплата:** Доступные методы зависят от настроек (**Заказы → Настройки → Методы оплаты**). Доступны «Наличные» и «Карта при получении». Онлайн-оплата временно недоступна — пункт в настройках заблокирован до подключения платёжного провайдера. При оплате наличными покупатель может указать, нужна ли сдача — сумма отображается в карточке заказа.

**Комментарий** — только при создании (не при редактировании)

После создания заказ появляется в соответствующей вкладке статуса.

## Редактирование заказа

На детальной странице заказа поля доступны для редактирования в зависимости от прав роли и **статуса заказа**. Изменения применяются кнопкой «Сохранить».

Пока заказ в группе «Новые» — редактируется всё: клиент, адрес, тип доставки, состав, оплата.

Как только заказ принят (группа «В работе») состав блокируется: существующие позиции нельзя менять или удалять — они уже могут готовиться. Это защищает кухню от рассинхронизации.

### Дозаказ блюда в принятый заказ

В принятый заказ доставки или самовывоза можно **дописать** блюдо (например, клиент перезвонил «добавьте ещё бургер»): под составом остаётся кнопка «Добавить». Существующие позиции при этом не трогаются — только добавление.

Если заказ уже ушёл на кухню, перед добавлением появится предупреждение: новая позиция уйдёт повару **отдельным тикетом** в очереди кухни. После подтверждения позиция появляется в составе, сумма заказа пересчитывается, а в «Истории» фиксируется событие «Добавлены позиции».

Изменить адрес или удалить уже добавленную позицию в принятом заказе пока нельзя — только дозаказ.

## Смена статуса

**Способ 1:** Кнопки быстрого перехода на карточке заказа. Переводят сразу в нужный статус одним кликом.

**Способ 2:** На детальной странице кнопка «Сменить статус» → выберите из списка.

### Запланированные заказы

Если у заказа указано время «к определённому времени» — меню смены статуса работает иначе: пока заказ висит в группе «Новые», оператору доступны только переход в «Запланировано» или отмена. Перейти сразу в «Принят» или «На кухне» нельзя — иначе кухня получит заказ раньше времени.

После перевода в «Запланировано» система сама переведёт заказ в рабочий статус (настраивается в **Заказы → Настройки → Предзаказы → Статус после срабатывания**), когда настанет время начала готовки.

## Заметки к заказу

На детальной странице — секция «Заметки». Добавляйте внутренние комментарии, которые видны только в панели управления, не клиенту. Полезно для пометок типа «клиент позвонил, просит подождать» или «адрес уточнён».

## История заказа

Секция «История» на детальной странице — хронологический лог всех изменений статусов с временными метками.

## История клиента

Секция «История клиента» — список предыдущих заказов этого же покупателя по телефону. Удобно чтобы понять, новый это клиент или постоянный.

## Массовые операции

В режиме таблицы — отметьте несколько заказов чекбоксами. Появится панель:
- **Сменить статус** — массовая смена всех выбранных заказов
- **Экспорт CSV** — скачать выбранные заказы как таблицу (имя, телефон, состав, сумма, тип, оплата, дата)

## Уведомление о новых заказах

Баннер «Появились новые заказы — нажмите, чтобы обновить» появляется в верхней части списка когда приходят новые заказы пока вы смотрите текущий список. Кликните на него чтобы обновить.
