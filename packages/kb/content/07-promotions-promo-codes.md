# Промокоды

> Требует включения модуля **Акции** в Настройки → Модули

Промокод — это строка которую покупатель вводит при оформлении заказа чтобы получить скидку. В отличие от акций, промокод нужно знать и ввести вручную.

## Когда использовать промокоды

- Рекламные кампании: «Введи SUMMER20 и получи скидку 20%»
- Программа лояльности: уникальный код для постоянных клиентов
- Партнёрские коллаборации: скидка для аудитории блогера
- Тестирование: дать код конкретным людям и отследить конверсию

## Создание промокода

**Акции → Промокоды → «+ Добавить промокод»**

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Код** | Да | Строка которую вводит покупатель. Показывается заглавными буквами |
| **Тип скидки** | — | Процент (%) или фиксированная сумма (₽) |
| **Размер скидки** | — | По умолчанию 10 |
| **Минимальная сумма заказа** | При фиксированной скидке | При %. — опционально. При ₽ — обязательно (не меньше размера скидки) |
| **Лимит использований** | Нет | Сколько раз код можно использовать всего. Пусто = без ограничений |
| **Активен с** | Нет | Дата начала |
| **Активен до** | Нет | Дата окончания |
| **Активен** | — | Тогл быстрого включения/выключения |

## Примеры кодов

| Код | Скидка | Назначение |
|-----|--------|-----------|
| `WELCOME` | 15% | Первый заказ нового клиента |
| `SUMMER25` | 25% | Летняя акция |
| `IVAN200` | 200 ₽ | Персональный код для клиента Ивана |
| `BLOGGER50` | 50% | Код для подписчиков блогера |

## Отслеживание использования

В таблице промокодов видно «X / Y» — сколько раз код использован из лимита. Когда лимит исчерпан — код автоматически деактивируется.

## Управление

- **Тогл «Активен»** — быстро включить/выключить
- **Редактировать** — изменить параметры
- **Удалить** — с подтверждением

> Если к промокоду привязан баннер — при отключении кода баннер перестанет работать.

## Советы

- **Понятный код** запоминается лучше: `PIZZA20` лучше чем `XK94RT`.
- **Верхний регистр** — стандарт: покупатель видит `SUMMER20`, не `summer20`.
- **Ограничьте по дате** для кампаний с дедлайном — код сам «выключится» когда нужно.
- **Лимит использований** полезен для ограниченных предложений: «Первые 100 заказов со скидкой».
- **Минимальный заказ** при фиксированной скидке — обязательно ставьте. Нельзя дать −500 ₽ при заказе от 100 ₽.
