# Модификаторы

> Требует включения модуля **Модификаторы** в Настройки → Модули

Модификаторы позволяют покупателю настроить блюдо перед добавлением в корзину: выбрать размер, степень прожарки, тип теста, уровень остроты и т.д.

## Когда использовать модификаторы

Используйте модификаторы когда:
- Одно блюдо продаётся в нескольких размерах (маленький / средний / большой)
- Есть варианты приготовления (средняя / хорошо прожаренная / с кровью)
- Нужен выбор основы (тонкое тесто / толстое / на цельнозерновой муке)
- Покупатель должен обязательно выбрать что-то (нельзя добавить без выбора)

**Пример:** Пицца → группа «Тесто» → опции «Тонкое», «Толстое», «Борт с сыром»

## Как это работает

1. Вы создаёте **группу** модификаторов (например «Размер») и добавляете в неё **опции** (S, M, L)
2. Привязываете группу к нужным **блюдам** — в форме каждого блюда
3. Покупатель на витрине видит переключатель и выбирает нужную опцию

## Создание группы модификаторов

Перейдите в **Меню → Модификаторы** и нажмите «+ Добавить группу».

### Поля группы

**Название группы** *(обязательно)* — отображается покупателю над опциями.
Примеры: «Размер», «Тесто», «Прожарка», «Острота».

**Активна** — тогл. Если выключена, группа и все её опции скрыты на витрине.

**Влияет на вес** — тогл. Включите если разные опции имеют разный вес.

Если **Влияет на вес** включён, появляется выбор режима:

| Режим | Когда использовать |
|-------|-------------------|
| **На уровне модификатора** | Вес одинаков для всех блюд, где используется группа. Указывается прямо здесь, рядом с опцией |
| **Каждое блюдо своё** | Вес задаётся отдельно для каждого блюда в форме блюда |

### Добавление опций

Каждая строка в списке — отдельная опция.

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Название опции** | Да | Например: «Маленький», «Тонкое», «Средняя» |
| **Вес** | Нет | Только если включён «Влияет на вес» + режим «На уровне модификатора» |

Нажмите «+ Добавить опцию» чтобы добавить новую строку. Минимум одна опция обязательна.

**Сортировка опций** — перетаскивайте строки за иконку ⠿. Порядок отображается покупателю именно так.

> **Важно:** Нельзя сохранить группу без хотя бы одной заполненной опции.

## Привязка к блюдам

Просто создать группу недостаточно — её нужно привязать к конкретным блюдам.

1. Откройте форму редактирования блюда
2. Перейдите в секцию «Модификаторы»
3. В списке «Доступные группы» нажмите «+ Добавить» рядом с нужной группой

После привязки настройте опции **для этого конкретного блюда**:

| Настройка | Описание |
|-----------|----------|
| **Чекбокс опции** | Включите чтобы опция была доступна для этого блюда |
| **+₽** | Добавочная цена за эту опцию (0 = бесплатно). Прибавляется к базовой цене блюда. Только 0 или больше — опция не может удешевлять блюдо |
| **Вес** | Только если группа в режиме «Каждое блюдо своё» |
| **По умолч.** | Кнопка-переключатель. Нажмите чтобы сделать эту опцию выбранной по умолчанию (только одна на группу). Подсвечивается синим когда активна |

**Пример:** Пицца «Маргарита», группа «Размер»:
- Маленькая (26 см) — +0 ₽, по умолчанию
- Средняя (30 см) — +150 ₽
- Большая (35 см) — +300 ₽

> **Копирование модификаторов.** В форме блюда кнопка «Скопировать из другого блюда» перенесёт все привязанные группы и настройки с другого блюда. Удобно для однотипных позиций.

## Управление группами

На странице «Меню → Модификаторы» вы видите таблицу со всеми группами.

- **Редактировать** — изменить название, опции, режим весов
- **Выключить** (тогл «Активна») — скрыть группу и все опции без удаления
- **Удалить** — удаляет группу вместе с привязками к блюдам. Система спросит подтверждение: «Группа «Размер» будет удалена. Привязки к блюдам тоже исчезнут.»

## Советы

- **Не создавайте группу на каждое блюдо.** Если у 10 пицц одинаковые размеры — создайте одну группу «Размер пиццы» и привяжите ко всем 10. При изменении цены достаточно поправить в одном месте.
- **Обязательный выбор** — если покупатель должен выбрать (например, размер), убедитесь что в группе не все опции опциональны. Включите хотя бы одну «по умолчанию».
- **Порядок важен.** Первая опция в списке — обычно базовая (самая дешёвая/маленькая). Покупатель видит их в вашем порядке.
- **Неактивная группа** vs **Удаление** — если группа временно не нужна, лучше выключить, а не удалять. Удаление уберёт все привязки к блюдам.
