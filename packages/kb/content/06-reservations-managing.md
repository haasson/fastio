# Работа с бронированиями

> Требует включения модуля **Бронирования** в Настройки → Модули

Бронирования поступают с витрины автоматически. Вы подтверждаете их, назначаете столы и управляете статусами.

## Список бронирований

На странице — все активные и предстоящие брони.

**Поиск** — по имени гостя или номеру телефона.

**Фильтр по дате** — выберите день, чтобы видеть только брони на него.

**Настройка колонок** — кнопка в тулбаре. Скройте ненужные колонки, выбор сохраняется.

## Создание брони вручную

Кнопка «+ Добавить» открывает форму.

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Имя гостя** | Да | Имя для идентификации |
| **Телефон** | Да | Формат +7 |
| **Дата** | Да | Список доступных дней с учётом расписания (выходные исключаются). Диапазон задаётся в настройках бронирований |
| **Время** | Да | Слоты по расписанию; прошедшие слоты для сегодняшней даты не показываются |
| **Количество гостей** | Да | От 1 до максимума (настраивается) |
| **Стол** | Да (если включён модуль «Столы») | Только столы текущего филиала с вместимостью ≥ количеству гостей |
| **Комментарий** | Нет | Особые пожелания гостя |

### Выбор стола при подтверждении

При подтверждении брони (статус «Ожидает») система показывает:
- **Доступные столы** — зелёным. Вместимость ≥ количеству гостей, нет пересечений с другими бронями на эту дату
- **Занятые столы** — оранжевым. Открыты сейчас или забронированы на это время

Если выбрать занятый стол — появится предупреждение.

|--------|----------------------|
| **Ожидает** | Бронь только поступила |
| **Подтверждена** | Вы нажали «Подтвердить» |
| **За столом** | Гость пришёл, нажали «Посадить» |
| **Завершена** | Визит окончен |
| **Не явился** | Гость не пришёл |
| **Отменена** | Бронь отменена (с указанием причины) |

## Обработка брони

Кликните на строку — откроется панель справа.

**Если статус «Ожидает»:**
- **Подтвердить** — сообщает гостю что всё ок
- **Отменить бронь** — запрашивает причину отмены

**Если статус «Подтверждена»:**
- **Посадить** — переводит в «За столом» и опционально открывает стол
- **Отменить бронь**

Если гость пришёл раньше времени — система предупредит сколько минут осталось до брони и спросит подтверждение.

## Архив

Завершённые, отменённые и несостоявшиеся брони — в разделе **Бронирования → Архив**.

Те же фильтры: дата, имя, телефон. Дополнительный фильтр по статусу (только «финальные» статусы).

Данные загружаются постранично — архив не ограничен по объёму.

## Уведомления о новых бронированиях

Счётчик в навигации показывает непросмотренные брони. Сбрасывается при заходе на страницу.

Email и Telegram-уведомления — настраиваются в **Настройки → Уведомления**.
