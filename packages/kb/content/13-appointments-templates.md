# Графики работы

Шаблоны графиков — это переиспользуемые расписания, которые применяются к сотрудникам и объектам в один клик.

**Запись → Графики работы.**

## Зачем нужны

Если у вас 10 мастеров с одинаковым расписанием — можно один раз создать шаблон «Утро» и применять к каждому новому сотруднику. Если потом измените шаблон — уже применённые расписания **не пострадают**, изменения подействуют только при новом применении (это сделано намеренно, чтобы изменение шаблона не ломало живые расписания).

## Как устроены слоты

Шаблон не задаёт время «открытия/закрытия» сам — он опирается на **график работы филиала**. В сетке вы видите все слоты, которые филиал работает в этот день, и просто отмечаете те, в которые сотрудник присутствует. Это значит: расширите часы филиала — никаких новых слотов в шаблоне не появится; сузите — лишние слоты при применении просто отрежутся.

Если у тенанта несколько филиалов с **разным** графиком — в дроере появится селектор филиала, чтобы выбрать референсную сетку. Если все одинаковые — селектор не показывается.

## Виды шаблонов

### По дням недели (weekly)

Семь дней — Пн..Вс. Для каждого дня отмечаете слоты, в которые сотрудник работает. Пустой день = выходной.

Хелпер «Скопировать в остальные» на понедельнике — копирует выбранные слоты во все остальные дни (только в те слоты, в которые филиал открыт).

Примеры:
- **Стандарт** — пн-пт все слоты филиала, сб-вс пусто
- **Утро** — пн-сб 08:00–13:00, вс пусто

### Сменный цикл (shift)

Для нестандартных циклов вида «2 утренних, 2 вечерних, 2 выходных». Параметры:
- **Длина цикла** — N дней (1..30)
- **Старт цикла** — конкретная дата, с которой считается «День 1»

Для каждого дня цикла отдельная сетка слотов (опирается на самое широкое окно работы филиала за неделю). Пустой день в цикле = выходной.

При применении шаблон раскатывается на диапазон дат: для каждой даты вычисляется индекс дня в цикле относительно стартовой даты, и в `resource_date_overrides` сотрудника падает копия слотов этого дня.

Примеры:
- **2/2 утро+вечер** — цикл 4 дня: День 1-2 = 09:00–13:00, День 3-4 = 17:00–21:00
- **Сутки через трое** — цикл 4 дня: День 1 = все слоты, дни 2-4 пусто

## Применение шаблона

В карточке сотрудника или объекта над недельным расписанием — кнопка «Применить шаблон». Выбираете шаблон, нажимаете «Применить» — текущее расписание заменяется. Для shift-шаблона дополнительно указываете диапазон дат.

Применение происходит атомарно: расписание + выключенные слоты + ссылка на шаблон записываются одной транзакцией; параметры расчёта (часы филиала, шаг слота) берутся из БД, не из формы — поэтому при двух одновременных применениях не получится «полу-применённого» состояния.

Изменения шаблона не подтягиваются автоматически. Хотите обновить расписание сотрудника — примените шаблон ещё раз.
