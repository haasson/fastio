# Настройки услуг

Раздел **Услуги → Настройки** — дефолтные значения для **новых** услуг. Меняете один раз, и каждая новая услуга создаётся уже с нужными настройками.

## Параметры

| Настройка | Описание |
|-----------|----------|
| **Запись онлайн** | Включена — новые услуги сразу доступны для онлайн-записи на витрине. Выключена — создаются как «только по звонку» |
| **Клиент выбирает исполнителя** | Включена — клиент сам выбирает мастера/объект. Выключена — система подбирает автоматически |
| **Тип записи** | **Фикс. длительность** или **Произвольная**. Подробнее — в разделе «Услуги» |
| **Максимальная длительность по умолчанию** | Только для произвольного типа. Верхняя граница, которую видит клиент, если у конкретной услуги не задан свой максимум |

## Применить к существующим услугам

При сохранении настроек система проверяет, у скольких услуг параметры отличаются от новых дефолтов. Если расхождения есть — появляется диалог с двумя вариантами:

- **Применить ко всем** — обновит настройки всех существующих услуг прямо сейчас. Нельзя отменить.
- **Только для новых** — сохраняет дефолты, не трогая существующие услуги.

Индивидуальные настройки каждой услуги всегда можно изменить в её карточке — дефолты не ограничивают.
