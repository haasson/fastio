# Модули

Модули — это функциональные блоки системы. Включайте только то что используете — лишние разделы в навигации только путают сотрудников.

## Доступные модули

| Модуль | Что добавляет |
|--------|--------------|
| **Доставка** | Приём заказов с доставкой на адрес, настройка зон и тарифов |
| **Самовывоз** | Приём заказов для самостоятельного забора |
| **Столы** | Управление столами, сессии, расчёт гостей |
| **Бронирования** | Предварительное бронирование столиков через витрину |
| **Кухня** | Экран очереди и сборки для поваров/упаковщиков |
| **Модификаторы** | Настройки блюд при заказе (размер, прожарка и т.д.) |
| **Добавки** | Дополнительные позиции к блюдам |
| **Акции** | Скидочные кампании, Happy Hour, промокоды |
| **Кастомные роли** | Создание ролей с гибкими наборами прав |

## Включение модуля

Тогл напротив модуля — мгновенное включение. Новый раздел появляется в навигации сразу.

## Отключение модуля

При отключении система проверяет есть ли активные зависимости. Если есть — покажет блокировку или предупреждение.

### Блокировки (нельзя отключить)

| Модуль | Блокирует если... |
|--------|-------------------|
| **Доставка** | Есть активные заказы на доставку |
| **Самовывоз** | Есть активные заказы на самовывоз |
| **Столы** | Есть открытые столы |
| **Бронирования** | Есть активные бронирования |
| **Кухня** | В очереди есть незавершённые позиции |
| **Кастомные роли** | Есть сотрудники с кастомными ролями |

Сначала завершите активные операции, потом отключайте.

### Предупреждения (можно отключить с подтверждением)

| Ситуация | Предупреждение |
|----------|---------------|
| **Акции** | Есть баннеры привязанные к акциям — они перестанут работать |
| Любой модуль | На витрине есть секция или страница завязанная на этот модуль — они пропадут |

При предупреждении система спрашивает подтверждение: вы понимаете последствия и продолжаете.

## Модули и тариф

Некоторые модули доступны только на определённых тарифах. Заблокированные модули показываются в секции «Недоступно на вашем тарифе» с иконкой замка.

Для доступа — обновите тариф в **Аккаунт → Биллинг**.

## Влияние на витрину

Когда вы отключаете модуль, связанные секции и страницы исчезают с витрины:
- Отключили «Доставку» → исчезает информация о доставке, зоны на карте
- Отключили «Бронирования» → исчезает страница бронирования

Чтобы вернуть — включите модуль и добавьте секцию заново в **Внешний вид**.
