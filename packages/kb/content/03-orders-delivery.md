# Зоны доставки

> Требует включения модуля **Доставка** в Настройки → Модули

Настройка зон доставки определяет куда вы доставляете и сколько это стоит. Есть два режима: единая стоимость для всех адресов или зоны с разными тарифами.

## Выбор режима

**Заказы → Доставка** — переключатель вверху страницы.

### Фиксированная стоимость

Один тариф для всех адресов. Подходит если вы работаете в небольшом радиусе без деления на районы.

| Поле | Описание |
|------|----------|
| **Минимальная сумма заказа** | При сумме ниже заказ не принимается. 0 = без минимума |
| **Стоимость доставки** | Цена в рублях. 0 = бесплатная доставка |
| **Бесплатная доставка от** | При заказе от этой суммы доставка бесплатна. 0 = всегда платная |

### Зоны доставки

Вы рисуете полигоны на карте и задаёте тарифы для каждой зоны. Покупатель вводит адрес — система сама определяет зону и показывает стоимость.

## Работа с картой

На карте видны:
- **Маркеры филиалов** — точки вашего заведения
- **Полигоны зон** — закрашенные области доставки

**Переключение между филиалами** — панель плиток сверху. Каждый филиал имеет свои зоны.

### Создание зоны

1. Нажмите «+ Нарисовать зону»
2. Кликайте на карте — ставьте точки границы зоны
3. Чтобы замкнуть полигон — кликните рядом с первой точкой (она подсветится)
4. Откроется панель настроек зоны справа

### Настройки зоны

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Название зоны** | Да | Например: «Центр», «Левый берег», «Промзона» |
| **Цвет** | — | Цвет полигона на карте |
| **Филиал** | Да (при нескольких) | К какому филиалу относится зона |
| **Стоимость доставки** | — | Цена доставки в эту зону. 0 = бесплатно |
| **Минимальный заказ** | — | Минимальная сумма для доставки в эту зону |
| **Бесплатная доставка от** | — | Порог бесплатной доставки для этой зоны |

### Редактирование зоны

Кликните на полигон зоны на карте — откроется панель с настройками. Измените параметры и нажмите «Сохранить».

### Удаление зоны

В панели настроек зоны — кнопка «Удалить».

## Важные предупреждения

> ⚠️ **Филиал без зон** — доставка для него не работает. Система покажет предупреждение «Нет зон доставки — доставка не работает» прямо в настройках филиала.

> ⚠️ **Адрес вне всех зон** — покупатель на витрине увидит сообщение «Адрес вне зоны доставки» и не сможет оформить заказ на этот адрес.

## Как видит это покупатель

При оформлении заказа на доставку:
1. Покупатель вводит адрес — появляются подсказки DaData
2. После выбора адреса система определяет зону
3. Отображается: стоимость доставки, минимальный заказ, порог бесплатной доставки
4. Если до бесплатной доставки осталось немного — показывается «До бесплатной доставки: 250 ₽»

## Советы

- **Перекрывающиеся зоны** — если адрес попадает в несколько зон, применяется та, которая была создана первой. Планируйте зоны без перекрытий.
- **Несколько филиалов** — система автоматически определяет ближайший филиал по адресу покупателя. Если адрес в зоне другого филиала — покупатель увидит уведомление.
- **Дорогостоящие зоны** — устанавливайте высокую стоимость доставки для удалённых районов вместо того чтобы исключать их. Так у покупателя остаётся выбор.
