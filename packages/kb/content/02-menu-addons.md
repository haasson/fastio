# Добавки

> Требует включения модуля **Добавки** в Настройки → Модули

Добавки — это дополнительные позиции, которые покупатель может добавить к блюду при оформлении заказа одним кликом. Отличие от модификаторов: добавки — отдельные позиции со своей ценой (соусы, напитки, десерты к основному блюду), а не варианты самого блюда.

## Когда использовать добавки

- Соусы к пицце или бургеру
- Дополнительный напиток к сету
- Десерт к основному блюду
- Хлеб к супу
- Дополнительные начинки за отдельную плату

## Структура

```
Добавки (раздел)
├── Конкретные добавки (Халапеньо, Соус барбекю, Кола...)
└── Пресеты (готовые наборы: "К пицце", "К бургеру"...)
```

**Добавки** — элементарные позиции с именем и ценой.

**Пресеты** — готовые наборы добавок для быстрой привязки к блюдам. Вместо того чтобы вручную выбирать 5 соусов для каждой из 15 пицц, создайте пресет «К пицце» и прикрепляйте его одним действием.

## Создание добавки

**Меню → Добавки → вкладка «Добавки» → «+ Добавить»**

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Название** | Да | Пример: «Халапеньо», «Соус барбекю», «Сырный соус» |
| **Цена, ₽** | Да | Целое число, минимум 0 |
| **Вес, г** | Нет | Целое число, минимум 0 |

Добавка автоматически становится активной. Чтобы скрыть её с витрины — используйте тогл «Активна» в списке.

## Создание пресета

**Меню → Добавки → вкладка «Пресеты» → «+ Добавить»**

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Название** | Да | Пример: «К пицце», «К бургеру», «Острый набор» |
| **Добавки** | Да, минимум 1 | Список добавок, входящих в пресет. Чекбоксы по каждой активной добавке |

## Глобальный лимит добавок

На странице **Меню → Добавки** вверху — переключатель «Ограничить количество добавок на блюдо».

Если включить — появляется числовое поле с максимальным количеством. Это ограничение действует для всех блюд по умолчанию.

Для конкретного блюда можно задать индивидуальный лимит — в форме редактирования блюда в секции «Добавки».

## Привязка к блюдам

Добавки работают только если привязаны к блюдам.

**Способ 1: Вручную**
1. Откройте форму редактирования блюда
2. Перейдите в секцию «Добавки»
3. Нажмите «+ Добавить добавку»
4. Выберите нужные из списка

**Способ 2: Через пресет**
1. В секции «Добавки» нажмите «+ Добавить добавку»
2. Выберите пресет — все его добавки добавятся сразу

**Способ 3: Скопировать с другого блюда**
Кнопка «Скопировать из другого блюда» в секции «Добавки» — перенесёт привязки с выбранного блюда в той же категории.

## Советы

- **Не перегружайте добавками.** 4–6 релевантных добавок работают лучше, чем 20 случайных.
- **Соусы — самые популярные добавки.** Если вы продаёте пиццу или бургеры — соусы почти всегда добавляют к заказу.
- **Используйте пресеты** если у вас много позиций одного типа. Создайте «К пицце», «К мясу», «К десертам» — и привязывайте пресет, а не каждую добавку отдельно.
- **Следите за активностью.** Добавка «Соус по-корейски» которой больше нет — скройте через тогл «Активна». Покупатели не смогут её выбрать, но данные сохранятся.
