# Настройки кухни

> Требует включения модуля **Кухня** в Настройки → Модули
> Требует права «Редактировать настройки»

Настройки определяют как заказы попадают на кухню и как кухня влияет на статусы заказов. Без правильной настройки кухонный модуль не работает.

## Статус-источник

**Когда заказ переходит в этот статус — все его блюда появляются в кухонной очереди.**

Обычно это статус «Принят» или «В работе». Как только оператор нажимает «Принять заказ» — повара сразу видят что нужно готовить.

> ⚠️ Если статус-источник не задан — кухонная очередь не будет получать новые блюда. Страница очереди покажет предупреждение.

## Статус при начале готовки

Необязательный. Если задан — заказ автоматически переходит в этот статус когда **повар берёт первое блюдо**.

Удобно чтобы оператор видел: заказ не просто принят, а уже готовится.

Можно не задавать если это различие для вас не важно.

## Статусы завершения

Когда сборщик нажимает «Собрано» — система должна знать в какой статус перевести заказ. Задаются отдельно для каждого типа доставки:

| Поле | Описание |
|------|----------|
| **Собрано: доставка** | Статус для заказов доставки. Например: «Передан курьеру» |
| **Собрано: самовывоз** | Статус для самовывоза. Например: «Готов к выдаче» |
| **Собрано: зал** | Статус для столовых заказов |

Поля показываются только если соответствующий модуль включён.

Если поле не задано — автоматическая смена статуса при сборке не происходит.

> ⚠️ Если не задан статус завершения для типа доставки которым вы пользуетесь — страница очереди покажет предупреждение.

## Порог срочности

**Минуты от 1 до 120.** По умолчанию: 15.

Управляет цветом таймера на карточках в очереди:
- Оранжевый: прошло **66%** от порогового времени
- Красный: прошло **100%** и больше

**Пример:** Порог 20 минут.
- 0–13 мин — нормально, без выделения
- 13–20 мин — оранжевый
- 20+ мин — красный и мигающий

Подберите значение исходя из реального времени приготовления в вашем заведении.

## Типичная конфигурация

```
Статус-источник:        Принят
Статус при готовке:     Готовится (опционально)
Завершение доставки:    Передан курьеру
Завершение самовывоза:  Готов к выдаче
Порог срочности:        15 минут
```

## Что происходит с блюдами при отмене заказа

Если заказ отменяется пока блюда в очереди — они помечаются как «Отменено» и появляются в отдельной секции у поваров и сборщиков. Нужно нажать «Убрать» чтобы подтвердить.

## Блюда без кухни

В форме каждого блюда есть тогл «Готовить на кухне». Если он выключен — блюдо не попадает в кухонную очередь даже при включённом модуле. Используйте для готовых блюд, напитков, выпечки которую не нужно готовить на месте.
