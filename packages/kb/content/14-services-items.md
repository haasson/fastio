# Услуги

Раздел **Услуги → Услуги** — каталог того, на что клиенты записываются: стрижки, процедуры, аренда столов и т.п.

## Создание услуги

Откройте категорию во вкладках сверху и нажмите «+ Добавить услугу». Если категорий нет — сначала создайте их в **Услуги → Категории**.

### Поля

| Поле | Описание |
|------|----------|
| **Название** | Видно клиенту на витрине |
| **Описание** | Свободный текст. Что входит в услугу, нюансы |
| **Цена** | Базовая стоимость. Может быть `0` для бесплатных консультаций |
| **Длительность** | В минутах. Используется для расчёта свободных слотов |
| **Категория** | К какой категории относится |
| **Фото** | Главное изображение для карточки и страницы услуги |
| **Активна** | Тогл. Выключенная услуга скрыта с витрины |

## Привязка к филиалам и ресурсам

После создания услуги — настройте, **где** и **кто** её выполняет:

- **Филиалы** — в каких филиалах эта услуга доступна. Если филиал один — выбирать не нужно.
- **Ресурсы** — каких исполнителей или объекты можно выбрать при бронировании. Связь редактируется в карточке услуги или в карточке ресурса (двунаправленно).

## Режимы бронирования

Каждая услуга может иметь свой режим:

- **Обычный** — клиент выбирает услугу, потом исполнителя/объект, потом время.
- **Без выбора исполнителя** — клиент выбирает только время, система автоматически подбирает свободного исполнителя.
- **Групповой** — несколько клиентов могут записаться на одно время (для занятий, мастер-классов).

## Управление услугами

**Сортировка** — перетаскивайте за иконку ⠿. Порядок внутри категории — это порядок на витрине.

**Включить/выключить** — тогл активности.

**Редактировать** — карандаш. Можно изменить любое поле.

**Удалить** — без ограничений на стороне услуги (старые записи на удалённую услугу остаются в истории, ссылка на услугу пустая).

## Советы

- **Дробите длинные услуги** на варианты разной длительности — клиенты увидят больше слотов в сетке.
- **Описание влияет на конверсию.** Опишите, что входит в услугу и сколько займёт.
- **Не запускайте без фото.** Карточка без изображения проседает в кликабельности.
