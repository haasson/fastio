# Услуги

Раздел **Услуги → Услуги** — каталог того, на что клиенты записываются: стрижки, процедуры, аренда столов и т.п.

## Создание услуги

Нажмите «+ Добавить услугу». Если категорий нет — сначала создайте их в **Услуги → Категории**.

### Основная информация

| Поле | Описание |
|------|----------|
| **Название** | Видно клиенту на витрине |
| **Описание** | Короткий текст. Что входит в услугу, нюансы |
| **Подробное описание** | Разворачивается по чекбоксу. Длинный текст для страницы услуги |
| **Цена** | Базовая стоимость. Может быть `0` для бесплатных консультаций |
| **Категория** | К какой категории относится |
| **Фото** | Главное изображение для карточки и страницы услуги |

### Настройки записи

| Настройка | Описание |
|-----------|----------|
| **Тип записи** | **Фикс. длительность** — услуга занимает известное время (стрижка 45 мин, массаж 60 мин). **Произвольная** — клиент сам выбирает длительность в слоттере (аренда корта, кабинета, снаряжения) |
| **Длительность** | Минут. Только для фиксированного типа |
| **Максимальная длительность** | Минут. Только для произвольного типа. Верхняя граница того, что клиент может выбрать. Если не задана — берётся из настроек раздела «Услуги → Настройки» |
| **Запись онлайн** | Включена — услуга появляется в форме записи на витрине. Выключена — только по звонку |
| **Клиент выбирает исполнителя** | Включена — клиент выбирает мастера/объект сам. Выключена — система подбирает автоматически |
| **Филиалы** | В каких филиалах доступна. Если филиал один — выбирать не нужно |
| **Активна** | Тогл. Выключенная услуга скрыта с витрины |

## Привязка исполнителей

В блоке «Запись онлайн» отображается список мастеров или объектов, привязанных к категории услуги. Если список пуст — клиент не сможет записаться. Перейдите в **Запись → Сотрудники / Объекты** и привяжите нужный ресурс к этой категории.

## Управление услугами

**Сортировка** — перетаскивайте за иконку ⠿. Порядок внутри категории — это порядок на витрине.

**Включить/выключить** — тогл активности.

**Редактировать** — карандаш. Можно изменить любое поле.

**Удалить** — без ограничений на стороне услуги (старые записи остаются в истории).

## Советы

- **Дробите длинные услуги** на варианты разной длительности — клиенты увидят больше слотов.
- **Описание влияет на конверсию.** Опишите, что входит в услугу.
- **Не запускайте без фото.** Карточка без изображения проседает в кликабельности.
- **Произвольный тип** подходит для аренды: клиент выбирает 1, 1.5 или 2 часа в одной форме.
