# Аккаунт и биллинг

## Профиль

**Аккаунт → Профиль** — информация о вашей учётной записи:

- **Имя** — отображается в интерфейсе
- **Email** — используется для входа
- **Роль** — ваша роль в заведении
- **Заведение** — название рабочего пространства

Страница только для просмотра. Для смены пароля нажмите «Изменить пароль» — на email придёт письмо со ссылкой.

**Выход из системы** — кнопка «Выйти» с подтверждением.

## Биллинг

> Доступно только владельцу заведения (право `billing.manage`)

### Текущий тариф

Показывает:
- Название плана
- Стоимость в месяц
- Дата следующего списания
- Баланс аккаунта

### Смена тарифа

Три тарифа отображаются в виде карточек с перечнем функций для вашего типа бизнеса:

| Тариф | Описание |
|-------|----------|
| **Витрина** | Только сайт, без заказов и записи |
| **Старт** | Базовый операционный функционал |
| **Про** | Полный функционал |

**Повышение тарифа** — доступно самостоятельно через кнопку «Перейти на [тариф]». Списывается с баланса.

**Понижение тарифа** — только через поддержку (кнопка неактивна с подсказкой).

**На пробном периоде** смена тарифа бесплатна — деньги не списываются, длительность триала не увеличивается. Над карточками показывается баннер с датой окончания пробного периода.

### Реактивация после блокировки

Когда подписка **приостановлена** (или просрочена), карточки тарифов переключаются в режим оплаты из баланса — без обращения в поддержку:

- На **текущем** тарифе — зелёная кнопка «Реактивировать · [сумма]».
- На **других** тарифах — кнопка «Подключить · [сумма]» (можно выбрать и более дешёвый — блок снимется по нему).
- Если баланса не хватает — кнопка неактивна с подписью «Недостаточно средств». Пополните баланс, и кнопки оживут (баланс обновляется в реальном времени).

По нажатию с баланса списывается стоимость выбранного тарифа, подписка становится активной на 30 дней, доступ восстанавливается сразу. Пополнение баланса само по себе блок не снимает — нужно подтвердить тариф кнопкой.

### История платежей

Список всех транзакций с датой, описанием и суммой.

### Статусы подписки

| Статус | Что происходит |
|--------|---------------|
| **Пробный период** | Полный доступ, 14 дней. После окончания нужно оплатить |
| **Активна** | Подписка оплачена, всё работает |
| **Приостановлена** | Доступ ограничен. Снять блок: пополнить баланс и реактивировать тариф на странице биллинга |
| **Отменена** | Подписка прекращена |

## Переключение между заведениями

Если вы владелец или участник нескольких заведений — в левом нижнем углу навигации кликните на название текущего заведения. Появится список доступных — выберите нужное.
