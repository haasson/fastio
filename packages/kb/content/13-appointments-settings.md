# Настройки записи

Раздел **Запись → Настройки** управляет поведением формы записи на витрине и обработкой заявок.

## Параметры

| Настройка | Описание |
|-----------|----------|
| **Название исполнителя** | Как называть исполнителей клиентам: «Мастер», «Тренер», «Специалист» и т.д. |
| **Горизонт бронирования** | На сколько дней вперёд клиент может записаться (по умолчанию 30 дней) |
| **Шаг слота** | Минимальная единица времени: 15, 30 или 60 минут. Влияет на сетку расписания |
| **Автоподтверждение** | Если включено — новые записи сразу получают статус «Подтверждена» без ручного подтверждения |
| **Отмена клиентом** | Разрешить клиентам самостоятельно отменять записи в личном кабинете |
| **Срок отмены** | За сколько часов до записи клиент может её отменить (работает только если отмена включена) |
| **Формат имени** | Как отображать имя исполнителя: полное, имя+инициал или только имя |

## Шаг слота и длительность услуги

Шаг слота определяет сетку: например, при шаге 30 минут слоты будут 09:00, 09:30, 10:00...

Длительность услуги задаётся в карточке услуги в разделе «Меню». Если услуга длится 90 минут, а шаг — 30 минут, то при записи на 09:00 следующий доступный слот будет не раньше 10:30.

## Автоподтверждение

При включённом автоподтверждении вам не нужно вручную подтверждать каждую запись — удобно для бизнесов без ручного контроля расписания.

При выключенном — каждая новая запись приходит со статусом «Новая» и требует вашего подтверждения. Счётчик в навигации покажет количество необработанных записей.

## Выбор исполнителя — настраивается на услуге

Параметр «Клиент выбирает исполнителя» теперь живёт на каждой услуге в разделе **Услуги** — потому что для одних услуг это критично (массаж — хочу к Свете), а для других нет (стрижка — без разницы). Включайте per-service.
