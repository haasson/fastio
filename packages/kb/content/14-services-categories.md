# Категории услуг

Категории — это разделы каталога услуг. Клиент видит их как вкладки или группы на витрине бронирования. Правильная структура делает выбор понятнее.

## Создание категории

Перейдите в **Услуги → Категории** и нажмите «+ Добавить категорию».

### Поля

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Название** | Да | Видно клиенту. Пример: «Стрижки», «Маникюр», «Массажи» |
| **Слаг для URL** | Нет | Используется в адресе страницы: `/category/<слаг>`. Только строчные латинские буквы, цифры и дефис. Подставляется автоматически |
| **Фото** | Нет | Квадратное изображение для навигации |
| **Цвет** | — | Цвет, которым подсвечиваются записи на таймлайне «Запись → Расписание». Карточка визита окрашивается в цвет категории её услуги — администратор с одного взгляда видит, какая категория идёт. При создании предлагается первый свободный цвет из палитры; занятые соседями приглушены, но кликабельны |
| **Активна** | — | Выключенная категория скрыта с витрины вместе со всеми услугами |

## Управление

На странице «Услуги → Категории» вы видите все категории в виде списка.

**Сортировка** — перетаскивайте за иконку ⠿. Порядок задаёт порядок вкладок на витрине.

**Включить/выключить** — тогл активности прямо в строке.

**Редактировать** — кнопка карандаша. Можно изменить название, фото, статус.

**Удалить** — только если в категории нет услуг. Если есть — переместите услуги в другую категорию.

## Советы

- **3–7 категорий** — оптимально для обзора без визуального шума.
- **Самые популярные — первыми.** Поставьте востребованные направления в начало.
- **Короткие названия** лучше длинных. «Маникюр» — да, «Маникюр и педикюр гель-лак» — нет.
- **Если услуги выполняют разные специалисты** — категории помогают клиенту фильтровать «Парикмахер» / «Бровист» / «Мастер маникюра».
