# Сборка

> Требует включения модуля **Кухня** в Настройки → Модули

Экран для сборщика или упаковщика. Показывает заказы целиком и помогает отслеживать готовность каждой позиции перед выдачей клиенту.

## Как устроен экран

Четыре колонки (фазы), через которые проходит каждый заказ:

| Фаза | Что значит |
|------|-----------|
| **Готовится на кухне** | Не все позиции заказа ещё приготовлены |
| **Требует сборки** | Все позиции готовы — нужно собрать и упаковать |
| **Готово** | Заказ собран, ждёт выдачи или курьера |
| **Отменено** | Отменённые заказы. Колонка видна только если они есть |

На мобильных устройствах колонки переключаются вкладками.

## Карточка заказа

На каждой карточке:
- Номер заказа
- Тип: «Доставка», «Самовывоз» или название стола
- Прогресс: «3/5» (готово 3 из 5 позиций)
- Список позиций с текущим статусом каждой

### Позиции из кухни (раздел «С кухни»)

Показывает блюда которые готовит кухня. Статус каждой:
- **В очереди** — ещё не взято поваром
- **Готовится** — повар взял в работу
- **Готово** — повар отметил как готовое
- **Подано** — выдано
- **Отменено** — позиция отменена

### Позиции для самостоятельной сборки (раздел «Собрать самому»)

Позиции которые не идут через кухню (например, напитки, готовые блюда). Сборщик сам отмечает их кликом на чекбокс.

## Процесс сборщика

1. Следи за колонкой **«Требует сборки»** — туда попадают заказы у которых все кухонные позиции готовы
2. Отмечай позиции по одной (чекбокс) или нажми **«Собрано»** на карточке
3. После «Собрано» заказ переходит в «Готово» и статус заказа автоматически обновляется

## Отменённые заказы

Когда заказ отменяется — карточка перемещается в «Отменено». Нажмите **«Убрать»** чтобы подтвердить и убрать карточку с экрана.

## Связь со статусами заказа

Когда сборщик нажимает «Собрано» — заказ автоматически переходит в нужный статус:
- Заказы доставки → статус завершения для доставки
- Заказы самовывоза → статус завершения для самовывоза
- Столовые заказы → статус завершения для зала

Эти статусы настраиваются в **Кухня → Настройки**.

## Советы

- **Сборка + выдача на одном экране.** Откройте «Сборка» на планшете рядом с зоной упаковки — сразу видно что готово.
- **Следите за «Требует сборки».** Как только заказ туда попадает — у вас есть только время до прихода курьера или клиента.
