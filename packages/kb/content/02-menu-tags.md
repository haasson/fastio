# Теги

Теги — это цветные значки, которые отображаются поверх фотографии блюда на витрине. Они помогают покупателю быстро ориентироваться: «острое», «новинка», «хит», «веганское».

## Как выглядит тег на витрине

Маленький цветной бейдж в углу карточки блюда: иконка + название. Тегов на одном блюде может быть несколько.

## Создание тега

**Меню → Теги → «+ Добавить тег»**

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **Название** | Да | Отображается рядом с иконкой на витрине. Подсказка-плейсхолдер зависит от типа бизнеса: «Острое» для общепита, «Новинка» для каталога, «Онлайн» для услуг |
| **Иконка** | Да | Набор иконок зависит от типа бизнеса: продукты и диета для общепита, товарные иконки для каталога, специализация для услуг. Плюс общие группы — маркетинг, время, статус |
| **Цвет** | Да | 14 цветовых палитр: красный, оранжевый, зелёный, синий, фиолетовый и др. |

В форме есть **превью** — вы сразу видите как тег будет выглядеть на витрине.

## Привязка тегов к блюдам

1. Откройте форму редактирования блюда
2. Перейдите в секцию «Теги»
3. Кликните на нужные теги — они подсветятся цветом

Один тег можно привязать к любому количеству блюд.

## Виртуальные категории на основе тегов

Тег можно использовать как основу для **виртуальной категории** — она автоматически собирает все блюда с этим тегом.

**Пример:** Создайте тег «Хит» и отметьте им популярные блюда. Затем создайте категорию «Хиты» типа «Виртуальная» с привязкой к тегу «Хит». На витрине появится раздел «Хиты» который всегда актуален — добавил тег к блюду, оно появилось в разделе.

Подробнее о создании таких категорий — в статье [Категории](/menu/menu-categories).

## Управление тегами

На странице «Меню → Теги»:

- **Сортировка** — перетаскивайте за ⠿. Порядок влияет на отображение когда у блюда несколько тегов
- **Редактировать** — кнопка карандаша
- **Удалить** — удаляет тег и убирает его со всех блюд (привязки тоже исчезают)

## Советы

- **2–4 тега в меню** — оптимально. Больше создаёт визуальный шум.
- **Используйте для маркетинга**: «Хит», «Новинка», «Ограниченное предложение» привлекают внимание лучше описания.
- **Диетические пометки** («Веганское», «Без глютена», «Халяль») помогают аудитории с ограничениями быстро находить подходящие блюда.
- **Не злоупотребляйте «Хитами»**: если у вас 30 блюд из 40 помечены как «Хит» — это теряет смысл. Хиты должны быть хитами.
- **Яркие цвета** для важных тегов: «Острое» — красный, «Новинка» — жёлтый. Нейтральные цвета (синий, серый) — для информационных пометок.
