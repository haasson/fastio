# Project Brief: Fastio

## Overview
FastFood SaaS — мультитенантная платформа для автоматизации ресторанного бизнеса. Включает панель администратора, витрину покупателя и публичную базу знаний.

## Core Requirements
- Мультитенантность: каждый ресторан — отдельный тенант со своими данными, филиалами, командой
- Realtime: заказы, кухня, столы — обновляются без перезагрузки
- Модульность: функциональность включается/выключается через TenantModules
- Роли: owner > admin > manager > staff с RLS на уровне базы данных

## Goals
- Дать ресторанам инструмент для приёма онлайн-заказов, управления меню, кухней, столами, бронированиями
- Обеспечить владельцу полный контроль через admin-панель
- Предоставить покупателю удобную витрину (storefront)
- Дать встроенную базу знаний (help) с AI-ассистентом

## Project Scope
**В скоупе:** admin SPA, storefront SSR, help SSR, Supabase backend, Edge Functions
**Вне скоупа:** мобильные приложения, внешние POS-системы
