# Product Context: Fastio

## Problem Statement
Рестораны нуждаются в единой платформе для: приёма онлайн-заказов, управления меню, кухонной очередью, столами, бронированиями, акциями и командой. Большинство решений либо слишком дорогие, либо не покрывают все нужды в одном месте.

## User Experience Goals
- **Владелец/Админ** — быстрая настройка, понятный интерфейс, всё в одном месте
- **Покупатель** — удобная витрина, быстрый заказ, понятный статус заказа
- **Кухня** — простая очередь заказов в реальном времени
- **Менеджер** — управление командой, филиалами, отчётность

## Apps
- `apps/admin` — Nuxt 3 SPA (ssr: false), порт 4710, панель администратора
- `apps/storefront` — Nuxt 3 SSR, витрина покупателя
- `apps/help` — Nuxt 3 SSR, порт 4712, публичная база знаний с AI-ассистентом
